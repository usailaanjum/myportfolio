First Website


The purpose of this website is to display JIT information relaying from a Particle Electron microcontroller attached to a moving vehicle. The collected location coordinates and speed are then sent to the Particle Cloud and JS is used to fetch the data with the Particle Cloud API and neatly displayed on a google map using a Google Maps API.

The website also has a database system to store user login and signup information system using MySQL and PHP.

This was my very first attempt at full-stack development with little prior knowledge. The project was completed as part of my 4th year Engineering Design Project.
